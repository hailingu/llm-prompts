# Waterfall Chart Template

## 概述
瀑布图（Waterfall Chart）用于显示连续的累积效应，直观展示数值变化的贡献。

## 适用场景
- 利润桥分析（Revenue Bridge）
- 现金流分析
- 成本结构变化
- 预算vs实际对比

## 视觉要素
- **起始柱**：基准值（蓝色）
- **增项柱**：正向贡献（绿色，向上）
- **减项柱**：负向贡献（红色，向下）
- **连接线**：虚线连接，显示累积路径
- **结束柱**：最终值（蓝色）

## 计算逻辑
```
结束值 = 起始值 + Σ(增项) - Σ(减项)
```
