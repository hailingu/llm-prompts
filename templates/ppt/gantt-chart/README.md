# Gantt Chart Template

## 概述
甘特图（Gantt Chart）由Henry Gantt于1910年发明，是项目管理中最常用的时间线可视化工具。

## 适用场景
- 软件开发项目
- 工程建设项目
- 产品发布计划
- 活动筹备时间表

## 核心元素
- **任务条**：横向长度=任务持续时间
- **里程碑**：关键节点（菱形标记）
- **依赖关系**：箭头连线
- **进度指示**：条形图填充比例
- **今日线**：垂直虚线标记当前日期

## 设计要点
- 任务按WBS分解结构组织
- 关键路径用红色高亮
- 最多显示15个任务（避免拥挤）
