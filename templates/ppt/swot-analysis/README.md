# SWOT Analysis Template

## 概述
SWOT分析是最广泛使用的战略规划工具，评估优势、劣势、机会、威胁四个维度。

## 适用场景
- 产品策略规划
- 市场进入决策
- 竞争对手分析
- 个人职业规划

## 四象限逻辑

| 内部/外部 | 积极因素 | 消极因素 |
|----------|---------|---------|
| **内部** | ✅ Strengths | ❌ Weaknesses |
| **外部** | ✅ Opportunities | ❌ Threats |

## 战略矩阵

- **SO战略**（优势+机会）：进攻策略
- **WO战略**（劣势+机会）：扭转策略
- **ST战略**（优势+威胁）：多元化策略
- **WT战略**（劣势+威胁）：防御策略
